# DNR Agri Exports — Website

This is a Next.js app scaffolded and customized for DNR Agri Exports.

Tech stack: Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion (client), Lucide React.

To run locally:

```bash
cd website
npm install
npm run dev
```

The project includes reusable components and pages for Home, About, Categories, Products, Quality, and Contact.
